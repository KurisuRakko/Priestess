# AGENTS.md

## 本目录职责
- 维护规则引擎、IP/UA/WAF 等策略判断逻辑。
- 重点是规则表达清晰、组合关系稳定、边界一致。

## 允许修改范围
- 允许修改规则实现、规则组合与本目录测试。
- 涉及策略触发后的业务后果时，优先交由 `object/` 或调用方处理。

## 必须复用的现有实现或组件入口
- 优先复用已有规则接口、组合器和缓存模式。
- 新增规则前先检查能否用现有组合与参数表达出来。

## 禁止事项
- 不要把具体控制器响应或 UI 行为写进规则层。
- 不要为了单个策略再复制一套判断框架。

## 本目录最小验证命令
- `go test ./rule/...`
- 如规则被多个对象使用，再补查相关对象测试。

## 与上级规则的继承或覆盖说明
- 继承根规则；本目录额外强调“策略表达统一”。
- 当前目录仅补充规则引擎侧约束。
