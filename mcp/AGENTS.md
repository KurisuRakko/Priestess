# AGENTS.md

## 本目录职责
- 维护 MCP 相关基础工具与协议辅助逻辑。
- 这里更偏协议适配或桥接层，不应承载大块业务。

## 允许修改范围
- 允许修改 MCP 工具函数和与协议处理直接相关的局部逻辑。
- 新增 MCP 能力时先保持与现有控制器和对象层接口衔接。

## 必须复用的现有实现或组件入口
- 优先复用已有协议工具和现有控制器入口。
- 需要共享对象能力时优先调用 `object/` 而不是重复实现。

## 禁止事项
- 不要在工具层塞入完整业务编排。
- 不要绕开现有协议入口自行创建第二套调用路径。

## 本目录最小验证命令
- `go test ./...`
- 如改动控制器协作链路，再补查 MCP 相关接口调用点。

## 与上级规则的继承或覆盖说明
- 继承根规则；本目录额外强调“协议层轻量化”。
- 当前目录只补充 MCP 工具约束。
