# AGENTS.md

## 本目录职责
- 维护表格相关前端组件与表格展示辅助逻辑。
- 重点是列表页表格体验、列定义复用和操作一致性。

## 允许修改范围
- 允许修改表格组件、列渲染、局部操作项和样式。
- 页面级筛选和业务逻辑仍优先留在页面文件。

## 必须复用的现有实现或组件入口
- 优先复用现有列表页表格模式和共享列渲染方式。
- 新增表格交互前先检查是否已有共用表格组件或工具。

## 禁止事项
- 不要把某个页面独有的大段业务逻辑沉到通用表格层。
- 不要复制已有表格行为只为改一个列显示。

## 本目录最小验证命令
- `cd web && yarn fix`
- 如影响多个列表页，可补跑 `cd web && yarn build`。

## 与上级规则的继承或覆盖说明
- 继承 `web/src/AGENTS.md`；本目录额外强调“列表交互统一”。
- 当前目录只覆盖表格共享层。
