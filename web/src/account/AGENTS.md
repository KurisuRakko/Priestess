# AGENTS.md

## 本目录职责
- 维护与账号、自助设置或个人资料相关的前端页面与局部交互。
- 重点是账号侧表单、展示状态和与用户自助流程的一致性。

## 允许修改范围
- 允许修改账号页面、局部状态处理和与账号 UI 直接相关的样式。
- 如涉及接口请求，优先通过 `web/src/backend/` 已有入口扩展。

## 必须复用的现有实现或组件入口
- 优先复用 `web/src/common/` 中的表单、选择器和反馈组件。
- 账号类页面布局尽量跟随仓库现有编辑页模式。

## 禁止事项
- 不要在账号页面里直接内联大段请求逻辑。
- 不要复制其他页面的整套表单结构只做字段替换。

## 本目录最小验证命令
- `cd web && yarn fix`
- 如改动用户自助流程，可补跑 `cd web && yarn test --watchAll=false`。

## 与上级规则的继承或覆盖说明
- 继承 `web/src/AGENTS.md`；本目录额外强调“账号自助流程一致性”。
- 仅在账号页面特例上覆盖上级。
