# AGENTS.md

## 本目录职责
- 维护基础页面、仪表盘、应用列表和通用首页级页面。
- 这里更接近全局入口和基础展示层。

## 允许修改范围
- 允许修改基础页面、布局、局部展示逻辑和与这些页面直接相关的样式。
- 需要共享组件时优先沉到 `common/`，不要在基础页面重复维护。

## 必须复用的现有实现或组件入口
- 优先复用现有卡片、网格、仪表盘和快捷入口组件。
- 新增基础页块时先搜索是否已有相似容器或布局。

## 禁止事项
- 不要把专属业务组件长期放在基础页面目录。
- 不要为了小改动复制整页布局。

## 本目录最小验证命令
- `cd web && yarn fix`
- 如改动首页或基础入口，再补跑 `cd web && yarn build`。

## 与上级规则的继承或覆盖说明
- 继承 `web/src/AGENTS.md`；本目录额外强调“基础页面尽量复用共享容器”。
- 仅覆盖基础展示层特例。
