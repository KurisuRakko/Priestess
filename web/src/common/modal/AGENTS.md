# AGENTS.md

## 本目录职责
- 维护可复用弹窗组件和弹窗交互封装。
- 重点是统一弹窗行为、关闭语义、确认反馈和表单容器风格。

## 允许修改范围
- 允许修改弹窗组件、弹窗内部表单和与弹窗行为直接相关的样式。
- 通用弹窗能力优先继续放这里，不要在页面里临时复制模态框实现。

## 必须复用的现有实现或组件入口
- 优先复用现有 modal 组件结构、按钮区布局和已有弹窗命名习惯。
- 需要确认、密码、裁剪、人脸等模式时先扩展相近组件。

## 禁止事项
- 不要在 modal 里塞入页面级复杂业务编排。
- 不要为一次性弹窗复制通用弹窗壳。

## 本目录最小验证命令
- `cd web && yarn fix`
- 高影响弹窗改动可补跑 `cd web && yarn build`。

## 与上级规则的继承或覆盖说明
- 继承 `web/src/common/AGENTS.md`；本目录额外强调“弹窗行为统一”。
- 仅在共享 modal 语义上覆盖上级。
