# AGENTS.md

## 本目录职责
- 维护商品、购物车等产品域共享前端组件。
- 重点是复用产品交互，不让商品 UI 在多个页面重复实现。

## 允许修改范围
- 允许修改产品共享组件、局部交互和与产品展示直接相关的样式。
- 业务定价规则仍优先由页面或后端接口决定。

## 必须复用的现有实现或组件入口
- 优先复用现有购物车控制组件和产品展示模式。
- 新增产品型组件前先搜索是否已有相近卡片或控制条。

## 禁止事项
- 不要把完整商品页逻辑沉到共享组件。
- 不要复制产品交互控件只做文案替换。

## 本目录最小验证命令
- `cd web && yarn fix`
- 如改动商品展示主流程，可补跑相关 Cypress 用例。

## 与上级规则的继承或覆盖说明
- 继承 `web/src/common/AGENTS.md`；本目录额外强调“产品域共享而非页面化”。
- 仅在产品共享组件层补充规则。
