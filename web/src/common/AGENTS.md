# AGENTS.md

## 本目录职责
- 维护前端共享组件、通用表单部件、通用业务小组件与局部样式。
- 这里是页面间复用的第一落点，职责是抽取真正稳定的共性。

## 允许修改范围
- 允许新增或修改共享组件、局部通用样式和可被多个页面复用的交互封装。
- 如果组件只服务单一页面且短期没有复用价值，优先留在原页面目录。

## 必须复用的现有实现或组件入口
- 优先复用现有 modal、select、theme、product、notification 子目录和已有通用组件命名方式。
- 抽公共前先确认仓库里没有功能等价组件。

## 禁止事项
- 不要把页面专属状态机伪装成通用组件。
- 不要为了抽象而抽象，制造难以追踪的 props 层。

## 本目录最小验证命令
- `cd web && yarn fix`
- 共享组件改动较广时补跑 `cd web && yarn build`。

## 与上级规则的继承或覆盖说明
- 继承 `web/src/AGENTS.md`；本目录额外强调“只抽稳定共性”。
- 子目录如有更近规则，优先使用子目录规则。
