# AGENTS.md

## 本目录职责
- 维护通知相关的共享前端组件。
- 重点是让通知能力在不同页面中表现一致。

## 允许修改范围
- 允许修改通知组件、通知触发辅助逻辑和局部样式。
- 如果通知逻辑依赖具体业务流程，优先把业务判断留在页面层。

## 必须复用的现有实现或组件入口
- 优先复用现有通知组件结构和已有消息反馈样式。
- 新增通知场景前先检查是否能扩展当前组件。

## 禁止事项
- 不要在通知组件中写入页面专属业务决策。
- 不要复制同一类通知展示。

## 本目录最小验证命令
- `cd web && yarn fix`
- 如改动影响较广，可补跑 `cd web && yarn build`。

## 与上级规则的继承或覆盖说明
- 继承 `web/src/common/AGENTS.md`；本目录额外强调“通知表现一致”。
- 当前目录只覆盖通知共享组件。
