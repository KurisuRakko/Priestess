# AGENTS.md

## 本目录职责
- 维护主题编辑、颜色和圆角等主题相关共享组件。
- 重点是把主题配置以统一、可预期的方式暴露给页面。

## 允许修改范围
- 允许修改主题组件、主题参数交互和局部样式。
- 如果涉及全局主题语义，优先检查 `web/src/theme` 或页面调用方式。

## 必须复用的现有实现或组件入口
- 优先复用现有颜色、半径、主题选择组件和现有 token 结构。
- 新增主题能力前先确认现有编辑器能否扩展。

## 禁止事项
- 不要把页面专属视觉 hack 塞进共享主题组件。
- 不要随意引入与现有主题字段不兼容的新结构。

## 本目录最小验证命令
- `cd web && yarn fix`
- 涉及主题主流程时再补跑 `cd web && yarn build`。

## 与上级规则的继承或覆盖说明
- 继承 `web/src/common/AGENTS.md`；本目录额外强调“主题参数结构统一”。
- 当前目录只覆盖共享主题组件。
