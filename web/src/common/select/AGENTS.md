# AGENTS.md

## 本目录职责
- 维护组织、语言、国家、主题等共享选择器。
- 重点是统一数据加载、值结构和交互体验。

## 允许修改范围
- 允许修改共享选择器、下拉数据映射和局部样式。
- 若数据来源变化，优先从 `backend/` 或已有工具层接入。

## 必须复用的现有实现或组件入口
- 优先复用现有选择器 props 约定、值格式和 Option 渲染模式。
- 新增选择器前先确认是否能在现有选择器基础上参数化。

## 禁止事项
- 不要把页面专属字段校验写在通用选择器里。
- 不要让同类选择器返回不同形状的值。

## 本目录最小验证命令
- `cd web && yarn fix`
- 如改动影响多页表单，可补跑 `cd web && yarn build`。

## 与上级规则的继承或覆盖说明
- 继承 `web/src/common/AGENTS.md`；本目录额外强调“选择器值结构稳定”。
- 当前目录只覆盖共享选择器特例。
