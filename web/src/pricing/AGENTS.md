# AGENTS.md

## 本目录职责
- 维护定价和价格展示相关前端页面或组件。
- 重点是金额、周期、计划展示与购买入口的一致性。

## 允许修改范围
- 允许修改定价展示、价格交互和与价格域直接相关的样式。
- 涉及实际计费接口时优先复用 `backend/` 和产品域已有逻辑。

## 必须复用的现有实现或组件入口
- 优先复用现有价格展示组件、商品组件和页面布局。
- 新增价格型展示前先搜索 `Pricing*`、`Product*` 相关页面。

## 禁止事项
- 不要把后端计费规则直接硬编码到前端展示层。
- 不要复制相近定价卡片结构。

## 本目录最小验证命令
- `cd web && yarn fix`
- 如改动购买主流程，可补跑相关 Cypress 支付或产品用例。

## 与上级规则的继承或覆盖说明
- 继承 `web/src/AGENTS.md`；本目录额外强调“价格展示一致与金额谨慎”。
- 仅覆盖定价前端特例。
