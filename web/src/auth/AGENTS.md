# AGENTS.md

## 本目录职责
- 维护登录、注册、回调、授权确认和第三方登录相关页面与流程组件。
- 这里承载认证入口，任何交互改动都要优先守住成功率与兼容性。

## 允许修改范围
- 允许修改认证页面、第三方登录按钮、回调处理、授权页面和局部样式。
- 接口细节优先复用 `AuthBackend`、`backend/` 和现有 provider button 模式。

## 必须复用的现有实现或组件入口
- 优先复用已有登录页结构、按钮组件、回调工具函数和错误提示方式。
- 新增认证入口先检查是否已有 provider 按钮或通用包装可扩展。

## 禁止事项
- 不要在认证页面里复制相同登录流程逻辑。
- 不要为了视觉调整破坏现有登录和回调稳定性。

## 本目录最小验证命令
- `cd web && yarn fix`
- 认证链路高风险改动时执行 `cd web && npx cypress run --spec cypress/e2e/login.cy.js`。

## 与上级规则的继承或覆盖说明
- 继承 `web/src/AGENTS.md`；本目录额外强调“认证成功率与流程稳定”。
- `mfa/` 下如有更近规则，优先使用子目录规则。
