# AGENTS.md

## 本目录职责
- 维护前端 E2E 测试目录的总体约束、测试分层和运行方式。
- 这里负责定义 Cypress 用例与支撑代码的组织边界。

## 允许修改范围
- 允许修改 Cypress 目录级约束、运行约定和对子目录的协作说明。
- 具体测试用例与支撑命令仍优先放到 `e2e/` 与 `support/`。

## 必须复用的现有实现或组件入口
- 优先复用现有 Cypress 命令、夹具思路和登录流程用例模式。
- 新增测试前先搜索是否已有同资源用例可以扩展。

## 禁止事项
- 不要把截图、视频产物当作源码维护。
- 不要在目录级文档里替代具体用例的断言约定。

## 本目录最小验证命令
- `cd web && npx cypress run`
- 如只改 support 或单类用例，至少能定位到对应测试文件并保证路径正确。

## 与上级规则的继承或覆盖说明
- 继承 `web/AGENTS.md` 与根规则；本目录额外强调“测试目录清晰分层”。
- `e2e` 与 `support` 下如有更近规则，以更近规则为准。
