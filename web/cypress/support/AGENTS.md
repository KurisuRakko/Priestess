# AGENTS.md

## 本目录职责
- 维护 Cypress 公共命令、全局钩子和测试支撑代码。
- 这里的职责是减少用例重复，而不是隐藏业务断言。

## 允许修改范围
- 允许修改自定义命令、全局 setup、测试辅助函数和支撑配置。
- 如果某段逻辑只服务单个资源流程，优先留在对应用例文件。

## 必须复用的现有实现或组件入口
- 优先复用现有 `commands.js`、`e2e.js` 和既有登录/导航辅助方式。
- 新增支撑函数前先确认是否已有近义命令可以扩展。

## 禁止事项
- 不要把具体业务断言封死在公共命令里。
- 不要让 support 层成为测试逻辑黑箱。

## 本目录最小验证命令
- `cd web && npx cypress run --spec cypress/e2e/login.cy.js`
- 如改动公共命令，至少挑一条高频用例验证。

## 与上级规则的继承或覆盖说明
- 继承 `web/cypress/AGENTS.md` 与上级规则；本目录额外强调“公共命令透明、可读”。
- 仅在测试支撑层覆盖上级。
