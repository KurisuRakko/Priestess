# AGENTS.md

## 本目录职责
- 维护与 MCP 自身行为相关的辅助逻辑和桥接代码。
- 重点是把内部能力以统一方式暴露给 MCP 流程。

## 允许修改范围
- 允许调整自包含 MCP 逻辑、适配层和与当前目录直接相关的帮助函数。
- 跨层能力仍应优先复用 `mcp/`、`controllers/` 和 `object/`。

## 必须复用的现有实现或组件入口
- 优先复用现有 MCP 工具与统一入口，不要另起协议分支。
- 需要领域数据时优先调用已有对象层能力。

## 禁止事项
- 不要在本目录复制完整协议实现。
- 不要让自用桥接逻辑偏离主 MCP 语义。

## 本目录最小验证命令
- `go test ./...`
- 如改动协议桥接路径，至少检查受影响接口可编译通过。

## 与上级规则的继承或覆盖说明
- 继承根规则；本目录额外强调“桥接而非分叉”。
- 仅在当前自用 MCP 语义上补充约束。
