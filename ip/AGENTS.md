# AGENTS.md

## 本目录职责
- 维护 IP 解析、地理库读取和地址工具逻辑。
- 重点是输入格式鲁棒性和解析结果稳定性。

## 允许修改范围
- 允许调整 IP 解析、查询接口和本目录测试。
- 跨层风控或规则使用方仍应在 `rule/` 或 `object/` 中表达。

## 必须复用的现有实现或组件入口
- 优先复用已有 IP 校验、错误定义和地理信息结构。
- 新增格式处理先参考现有 IPv4/IPv6 兼容方式。

## 禁止事项
- 不要把风控业务决策硬编码在底层 IP 工具里。
- 不要复制相同的解析与校验流程。

## 本目录最小验证命令
- `go test ./ip/...`
- 如影响规则判断，再补跑 `go test ./rule/...`。

## 与上级规则的继承或覆盖说明
- 继承根规则；本目录额外强调“底层工具纯粹性”。
- 当前目录仅补充 IP 工具特例。
