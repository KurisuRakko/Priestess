# AGENTS.md

## 本目录职责
- 维护代理相关的核心实现与转发能力。
- 重点是请求转发行为、客户端初始化与代理边界稳定。

## 允许修改范围
- 允许修改代理实现、代理辅助函数和与代理行为直接相关的局部逻辑。
- 如果代理逻辑依赖站点规则，优先复用 `service/` 与 `object/site*` 现有能力。

## 必须复用的现有实现或组件入口
- 优先复用现有 HTTP 客户端初始化、错误处理和站点映射规则。
- 不要自行创建平行代理入口。

## 禁止事项
- 不要把站点业务策略散落在代理底层。
- 不要复制请求改写和重试逻辑。

## 本目录最小验证命令
- `go test ./...`
- 如影响站点代理链路，再补查 `service/` 相关调用。

## 与上级规则的继承或覆盖说明
- 继承根规则；本目录额外强调“代理语义一致”。
- 仅在代理实现层覆盖上级。
