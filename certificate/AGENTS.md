# AGENTS.md

## 本目录职责
- 维护证书生成、解析、校验和相关安全材料处理。
- 这里的变更通常影响 TLS、签名或证书导入导出行为。

## 允许修改范围
- 允许调整证书工具函数、格式转换和与证书对象直接相关的测试。
- 跨层证书生命周期逻辑优先仍放在 `object/`。

## 必须复用的现有实现或组件入口
- 优先复用现有证书处理函数、测试样例和文件格式约定。
- 涉及证书落库或对象绑定时优先跟随 `object/cert*.go` 既有模式。

## 禁止事项
- 不要把业务流程编排或控制器响应格式塞进本目录。
- 不要随意改动现有证书格式假设而不检查兼容性。

## 本目录最小验证命令
- `go test ./certificate/...`
- 如变更影响对象层证书能力，再补跑 `go test ./object/...`。

## 与上级规则的继承或覆盖说明
- 继承仓库根目录规则；本目录额外强调“安全材料兼容性优先”。
- 当前目录仅覆盖证书工具层特例。
